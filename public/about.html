Идея и назначение

Небольшой учебный web‑сервис для демонстрации полного стека: HTML, CSS, JavaScript, Express и MongoDB.

Зачем нужен TaskTrackr

Сервис помогает систематизировать учебные задачи: от подготовки к ЕГЭ до курсовых и пет‑проектов. Здесь можно создавать задачи, задавать дедлайны, отслеживать статусы и смотреть сводную статистику.

Проект специально спроектирован так, чтобы закрыть требования лабораторной работы: семантическая разметка, адаптивная вёрстка, интерактивность на JavaScript и работа с REST API и базой данных.

Технологический стек

Фронтенд
HTML5, CSS3 (Flexbox, Grid, media queries), JavaScript ES6+
Бэкенд
Node.js, Express.js, REST API (CRUD по задачам)
База данных
MongoDB + Mongoose, схема задачи с валидацией
Инструменты
Git, Chrome DevTools, Postman/cURL, MongoDB Compass

Структура сайта

Набор взаимосвязанных страниц, демонстрирующих разные аспекты веб‑разработки.

  • Главная — создание и список задач, слайдер, работа с JSON и XML, интерактивность на JavaScript.
  • Статистика — агрегированные данные по задачам, таблицы с вложенными таблицами и якоря.
  • О сервисе — описание архитектуры и технологий, список определений и документация.
